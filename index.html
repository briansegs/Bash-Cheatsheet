Bash Cheat sheet

VARIABLES
varname=value defines a variable.
echo $varname checks a variable's value.
Basic Terminal Navigation
ls lists your files in current directory, ls <dir> to print files in a specific directory.
ls -l lists your files in 'long format', which contains the exact size of the file, who owns the file and who has the right to look at it, and when it was last modified.
ls <folderName> list files in folder.
*(patternlist) matches zero or more occurrences of the given patterns.
cd changes to home.
cd .. changes to the parent directory.
cd <folderName> changes directory.
pwd print working directory.
Basic file manipulation
mkdir create new folder.
mv <filename> <dir>/ move a file to a directory.
rm <filename> delete file.
rm -i <filename> ask for confirmation before deleting file.
rmdir <dirname> remove an empty directory.
Downloading (curl)
curl <'web address'> downloads a file from the web. By default, it will download the file and display the contents in the terminal.
curl -L <'web address'> The same as curl but, follows redirects.
curl -o <filename> <'web address'> The same as curl but, outputs the data to a file. can be used with -L.
Viewing files (cat, less)
cat <filename> displays the full contents of the file.
less <filename> displays the output in a format that allows you to search and scroll.
spacebar or down arrow = scroll down. b key or up arrow = scroll up. forward slash / = search. q key = exit.
